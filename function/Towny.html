Towny 插件指令

指令 描述
/resident显示玩家的居民信息。
/res ?显示居民指令的帮助信息。
/res friend add [玩家名]添加好友，使其获得你的土地权限。
/res friend remove [玩家名]移除好友。
/res set perm设置个人地皮权限。
/town显示玩家所在城镇的信息。
/town ?显示城镇指令的帮助信息。
/town list [页码]列出所有城镇。
/town online显示在线的城镇居民。
/town [城镇名]显示指定城镇的信息。
/town leave离开当前城镇（居民指令）。
/town deposit [$金额]向城镇银行捐款。
/town withdraw [$金额]从城镇银行取款（城主指令）。
/town new [城镇名]创建新的城镇。
/town delete [城镇名]删除指定城镇（管理员指令）。
/town rank add [玩家名] [头衔名]为玩家添加城镇头衔。
/town rank remove [玩家名] [头衔名]移除玩家的城镇头衔。
/town set mayor [居民名]任命新的城主。
/town set homeblock设置城镇的主地块。
/town set spawn设置城镇的出生点。
/town set name [新城镇名]更改城镇名称。
/town set tag [标签]设置城镇标签。
/town toggle pvp切换城镇的 PvP 设置。
/town toggle public切换城镇是否公开。
/town toggle explosion切换城镇的爆炸设置。
/town toggle fire切换城镇的火焰蔓延设置。
/town toggle mobs切换城镇的怪物生成设置。
/town claim声明当前区块为城镇领地。
/town unclaim放弃当前区块的城镇领地。
/town claim outpost声明前哨基地。
/town unclaim outpost放弃前哨基地。
/town outpost list列出城镇的前哨基地。
/town outlaw add [玩家名]将玩家列为城镇的放逐者。
/town outlaw remove [玩家名]移除玩家的放逐者状态。
/town outlawlist查看城镇的放逐者列表。
/town say [消息]向城镇居民广播消息。
/nation显示玩家所在国家的信息。
/nation ?显示国家指令的帮助信息。
/nation list [页码]列出所有国家。
/nation online显示在线的国家成员。
/nation [国家名]显示指定国家的信息。
/nation leave离开当前国家（城主指令）。
/nation deposit [$金额]向国家银行捐款。
/nation withdraw [$金额]从国家银行取款（国王指令）。
/nation new [国家名]创建新的国家。
/nation delete [国家名]删除指定国家（管理员指令）。
/nation rank add [玩家名] [头衔名]为玩家添加国家头衔。
/nation rank remove [玩家名] [头衔名]移除玩家的国家头衔。
/nation ally add [国家名]添加盟国。
/nation ally remove [国家名]移除盟国。
/nation enemy add [国家名]添加敌国。
/nation enemy remove [国家名]移除敌国。
/nation set king [居民名]任命新的国王。
/nation set capital [城镇名]设置国家的首都。
/nation set taxes [$金额]设置国家税收。
/nation set name [新国家名]更改国家名称。
/nation set tag [标签]设置国家标签。
/nation toggle neutral切换国家的中立状态。
/nation say [消息]向国家成员广播消息。
/plot显示地皮信息。
/plot claim购买一块地皮。
/plot unclaim放弃一块地皮。
/plot set perm设置地皮权限。
/plot set name [名称]为地皮设置名称。
/plot set shop将地皮设置为商店类型。
/plot set embassy将地皮设置为大使馆类型。
/plot set arena将地皮设置为竞技场类型。
/plot set farm将地皮设置为农场类型。
/plot set inn将地皮设置为旅馆类型。
/plot set jail将地皮设置为监狱类型。
/plot toggle pvp切换地皮的 PvP 设置。
/plot toggle explosion切换地皮的爆炸设置。
/plot toggle fire切换地皮的火焰蔓延设置。
/plot toggle mobs切换地皮的怪物生成设置。
